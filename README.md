.git에 node_modules는 올리지 않는다

node_modules가 없으면 실행되지 않으나 npm restore만 하면 다시 실행 가능하다.
package.json만 있으면 가능

npx create-react-app appname -> 리액트 구현

npm start -> 서버 시작
